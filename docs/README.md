# Documentation

This directory contains project documentation.

## Available Documentation

- **[AGENT_INSTRUCTIONS.md](./AGENT_INSTRUCTIONS.md)** - Comprehensive guide for LLM coding agents working on this project. Includes development setup, common issues, patterns, and key learnings.
- **[ENTITY_ADDITION_RUNBOOK.md](./ENTITY_ADDITION_RUNBOOK.md)** - Step-by-step guide for AI agents to add entities to TahOak Park Collective using direct database insertion.
- **[ENTITY_DATA_COLLECTION_FORM.md](./ENTITY_DATA_COLLECTION_FORM.md)** - Form template for collecting entity information when adding new entities.
- **[ENTITY_TYPES_AND_CATEGORIES.md](./ENTITY_TYPES_AND_CATEGORIES.md)** - Reference for valid entity types and category combinations.
- **[AGENT_CONTEXT.md](./AGENT_CONTEXT.md)** - Quick reference for development context, database structure, and common commands.

## Documentation Standards

- Keep user-facing documentation in the root `README.md`
- Keep agent-specific and technical documentation in this `docs/` directory
- Update documentation when making significant architectural changes
- Include code examples where helpful
